# reveal-pon

reveal.js build system

# Usage

1. Click **Use this template** on GitHub
2. edit **src/slide.md**
3. commit and push then run ci automatically.
4. enable GitHub Pages (Settings > GitHub Pages > Source : select gh-pages)
5. edit README.md(optional)
